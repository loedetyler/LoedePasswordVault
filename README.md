# Loede Password Manager
Here at Loede Enterprises, we take password security seriously. That's why we developed a password manager to help you remember your passwords easier and make more complex passwords.


# Installation (If you want to run locally)

Clone this github repository <br>
Make sure Python is installed <br>
Navigate to the folder where this was cloned <br>
Install Django if it isn't already installed<br>
Run py manage.py runserver<br>
Navigate to localhost<br>


# Getting Started
To run my top of the line password vault, simply navigate to http://100.31.44.22:8000/<br>
From there you will see the home page.<br>
Click on the user icon in the top right and register an account.<br>
After registered, you will then be able to see the Vault option in the burger menu dropdown.<br>
Clicking on that will lead you to the Vault where you can start adding your passwords.<br>
